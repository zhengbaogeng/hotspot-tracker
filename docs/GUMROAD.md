# Hotspot Tracker - 热点追踪工具

## 产品描述

**Hotspot Tracker** 是一款自动化热点追踪工具，帮助内容创作者、自媒体运营者、市场分析师快速获取全网热点数据。

---

## 核心功能

### ✅ 免费版（v1.0.0）

| 功能 | 说明 |
|------|------|
| 多平台抓取 | 微博热搜、知乎热榜、头条热文 |
| 热度排名 | 自动按热度排序 |
| 数据输出 | JSON格式，便于二次开发 |
| 报告生成 | Markdown格式，可直接阅读 |
| 定时执行 | 支持cron定时任务 |

### 🔮 付费版（v1.1.0）- 99元/月

| 功能 | 说明 |
|------|------|
| 趋势预测 | 预测热点未来走势 |
| 生命周期分析 | 分析热点持续时间 |
| 创作建议 | 根据热点推荐创作方向 |
| 传播路径 | 分析热点传播路径 |
| API接口 | 提供API调用服务 |

---

## 使用场景

- **内容创作者** - 快速获取热点，创作爆款内容
- **自媒体运营** - 每日热点追踪，提升内容质量
- **市场分析师** - 热点数据分析，洞察市场趋势
- **品牌营销** - 热点营销机会，提升品牌曝光
